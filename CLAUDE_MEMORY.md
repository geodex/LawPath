## IMPORTANT PATTERNS
- **Multi-tenant isolation:** every table has `tenant_id uuid not null`. All queries scoped with `WHERE tenant_id = $1`.
- **Super admin:** `tenant_id = null` in users table. Controls platform-level settings only.
- **API fallback:** all API calls in frontend fall back to local state if backend unreachable (enables demo/dev without DB).
- **CSS variables:** `--ink, --muted, --line, --paper, --panel, --green, --green-dark, --gold, --blue, --rose` — dark mode overrides these in `@media (prefers-color-scheme: dark)`.
- **Git remote:** HTTPS (not SSH). Windows Credential Manager handles auth automatically.
- **PUPPETEER_SKIP_DOWNLOAD=true** must be set during `npm ci` on the server to avoid downloading bundled Chrome.
